Web Tiket Konser BMTH | Lomba EPIM TI 2024

Projek website ini dibuat untuk lomba web programming EPIM TI 2024 yang diadakan di Politeknik Negeri Jember.Website ini dibuat dengan HTML, CSS, Js, dan PHP native yang merupakan project team, dimulai dengan mendesain web secara matang di Figma lalu di slicing ke dalam kode.Fitur yang tersedia pada website ini antara lain : login, register, reset password, form kontak, selecting tiket, select metode pembayaran, dan yang terakhir yaitu payment.

Website Kumpulan Game-game HTML

Projek website ini saya buat untuk keperluan pribadi, yaitu sebuah web untuk menampung beberapa game-game yang saya buat menggunakan HTML, CSS, dan Js.Untuk fitur-fitur yang tersedia pada website ini yaitu selecting game untuk memudahkan pengguna dan form kontak agar user dapat memberikan masukannya.Website ini saya buat menggunakan Tailwindcss dan untuk back-end nya saya menggunakan Laravel 11.

Website Djember Fest | Lomba Play IT 2024

Website ini dibuat untuk keperluan lomba yang diadakan di Politeknik Negeri Malang. Djember Fest adalah website yang dibuat untuk memperkenalkan kearifan kota Jember yang sangat kaya akan budaya, destinasi, dan UMKM nya yang lezat.Fitur yang tersedia pada website ini antara lain : galeri budaya, page destinasi, page makanan, dan form kontak.Website ini dibuat menggunakan React Js dan merupakan project team.
